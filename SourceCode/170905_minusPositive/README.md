#### 코드 설명

함수는 아직 배우지 않았으므로 함수를 따로 만들지 않고 코드를 짰다.

데이터 타입이 Int인 두개의 변수와 두개의 값을 뺏을 때의 값인 totalVal를 데이터 타입 Int로 선언했다. 

조건문 if를 통해 minusNum1과 minusNum2를 뺀 값이 0보다 작은지 비교를 해서 0보다 작다면 각각의 변수에 ***-1***을 곱해서 값을 출력했다.
