#### 코드 설명

함수는 아직 배우지 않았으므로 함수를 따로 만들지 않고 코드를 짰다.

우선 절대값인지 아닌지 그 값을 받아야 하므로 데이터타입이 Int인 testNum변수를 선언했다.

그 이후 조건문 if를 통해 변수 testNum이 0보다 작은지를 판별해서 0보다 작을 경우에는 testNum에 -1을 곱해서 양수가 나오도록 했다.
